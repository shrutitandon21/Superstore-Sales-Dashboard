# Superstore Sales Dashboard – Power BI

## 📊 Project Overview
This project presents an **interactive Power BI dashboard** built using the **Superstore Sales dataset**. The report provides insights into sales performance, profitability, shipping efficiency, and future sales trends through data visualization and forecasting.

The dashboard is designed to help users quickly analyze key business metrics and make data-driven decisions.

---

## 🚀 Features
- Interactive Power BI report
- Two dashboard pages for analysis and forecasting
- Region-wise slicer for dynamic filtering
- Top N filters applied for focused analysis
- Multiple visualizations for better data storytelling

---

# 📁 Dashboard Pages

## 1️⃣ Sales Dashboard

The **Sales Dashboard** provides an overview of business performance through KPIs and visual analytics.

### Key Performance Indicators (KPIs)
- **Total Sales**
- **Total Profit**
- **Average Ship Days**
- **Total Quantity Sold**

### Visualizations Included
- 🗺 **State-wise Sales Map** – Geographic distribution of sales and profit
- 🍩 **Donut Chart – Sales by Region**
- 🍩 **Donut Chart – Sales by Payment Mode**
- 📊 **Column Chart – Sales by Segment**
- 📊 **Bar Chart – Sales by Ship Mode**
- 📊 **Bar Chart – Sales by Category**
- 📊 **Bar Chart – Sales by Sub-Category**
- 📈 **Stacked Area Chart – Monthly Sales (Year-over-Year)**
- 📈 **Stacked Area Chart – Monthly Profit (Year-over-Year)**

### Filters
- **Region-wise slicer** for dynamic filtering
- **Top N filters** to highlight top-performing segments

---

## 2️⃣ Sales Forecast Dashboard

The **Forecast Dashboard** focuses on predicting future sales trends and analyzing sales distribution across states.

### Visualizations Included
- 📈 **Line Chart – Sales Forecast for the Next 15 Days**
- 📊 **Bar Chart – State-wise Total Sales**

### Filters
- **Region slicer** for interactive filtering
- **Top N filters** for focused insights

---
